# Regular Expressions Solutions

## Username

`[a-zA-Z][\w]{3,8}[a-zA-Z]`

## Real name

`[^\\|!"@#£$§%&\/()=?{[\]\}'«»\*\+0-9]+`

## ZIP Code

`([\d]{4})(-[\d]{3})?`

## IPv4 Address

`^((25[0-5]|(2[0-4]|1[0-9]|[1-9]|)[0-9])(\.(?!$)|$)){4}$`

## MAC Address

`(([A-F0-9]{2}:){5}|([A-F0-9]{2}-){5})([A-F0-9]{2})`

## Password

`(?=.*[0-9].*[0-9])(?=.*[;\.:].*[;\.:])([\w;\.:])[\w;\.:]{4,}(?!\1)[\w;\.:]`
